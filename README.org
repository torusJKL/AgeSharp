* AgeSharp

C# implementation of the [[https://github.com/str4d/age][Age]] file encryption format.

=Age= is a simple, modern and secure file encryption tool.

*Note: This is not affiliated with the official Age project.*

** Prerequisites

- .NET 10 SDK

** Build

#+begin_src shell
dotnet build src/AgeSharp.Core/AgeSharp.Core.csproj
#+end_src

** Test

#+begin_src shell
dotnet test tests/AgeSharp.Tests/AgeSharp.Tests.csproj
#+end_src
